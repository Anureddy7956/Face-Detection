## Libraries required to be installed for Face Detection
1. OpenCV
   <pre>
     pip install opencv-python   
   </pre>
2. face_recognition
   <pre>
     pip install face_recognition 
   </pre>

3. dlib
   <pre>
     pip install dlib 
   </pre>

4. numpy
   <pre>
     pip install numpy 
   </pre>
